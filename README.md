# RICK AND MORTY APP

Aplicación que permite listar los personajes de la serie Rick and Morty, obteniendo el listado de personajes por medio del consumo de la API publica disponible en GraphQL. permitiendo seleccionar cada personaje para obtener más detalle y poder seleccionarlo como favorito.

# Consumo de API GRAPHQL

- Para el consumo de la API GRAPHQL se utilizó la libreria Apollo Client, creando dentro de la carpeta gql, el archivo base de configuración y exportando el cliente de conexión, se utiliza el provider para poder hacer el consumo de la API en toda la aplicación.
